# 🛡️ NEXUS-E SOVEREIGN: OMNISCIENCE (v22.0)

**"Verinin Sınırlarını Aşan, Geleceği Yöneten Otonom Finansal Strateji Merkezi"**

---

## 🧬 Proje Vizyonu
E-ticaret ve KOBİ finansı dünyasında, veriyi pasif bir kayıttan proaktif bir stratejik akla dönüştürmek. **NEXUS-E SOVEREIGN**, sadece bir dashboard değil; **Hesaplamalı Sosyal Bilimler (CSS)** metodolojisini kullanarak pazarın sosyal ve finansal entropisini yöneten otonom bir "Executive Partner"dır.

## 🚀 Neden 1. Olmalıyız? (Efsanevi Özellikler)
*   **Diferansiyel Volatilite Motoru:** Rakiplerin aksine, bütçe değişimlerini tüm ürünlere doğrusal değil; senaryo bazlı (kaotik/gerçekçi) olarak uygular.
*   **Neural Strategy Core:** Gemini 1.5 Flash tabanlı AI, KOBİ'ler için otonom icra talimatları ve EBITDA projeksiyonları üretir.
*   **CSS Entegrasyonu:** Sosyolojik verileri finansal stratejilerle harmanlayan ilk otonom karar mekanizması.
*   **Executive Reporting:** Tek tıkla CEO düzeyinde, profesyonel mizanpaja sahip (.doc) stratejik raporlar oluşturur.

## 🛠️ Teknik Mimari
*   **Engine:** Python 3.10+
*   **Intelligence:** Google Gemini 1.5 Flash API
*   **Framework:** Streamlit (High-Contrast Prism UI)
*   **Analytics:** Pandas, Numpy, Plotly

## 📦 Kurulum ve Çalıştırma
1. `.env` dosyasına `GEMINI_API_KEY` değerinizi girin.
2. `pip install -r requirements.txt` komutuyla kütüphaneleri yükleyin.
3. `streamlit run app.py` komutuyla sistemi başlatın.
